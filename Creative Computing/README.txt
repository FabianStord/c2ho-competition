Sonification of the Autoencoder learning process

Our artwork is a sonification made from images creating during the training process of an Autoencoder.

How to run:

Import all the packages needed
Specificy the image you want to use for training in line 3
Run the model for as many epochs as you want.
Crop all the images 
Create the .gif file from the images
Run the functions used for sonification
Sonify the images
Combine the .gif file with the audio
